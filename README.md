# Golang Fullstack

> a boiler plate for fullstack golang

## Pre-requisties

- Clone This Repository

```shell
git clone https://github.com/ilits2024/ilits2024-backend.git
```

- Install Dependencies

```shell
npm i
```

## Getting Started

- Build App

```shell
npm run build
```

- Run in Development Env

```shell
npm run dev
```

- Run in Production Env

```shell
npm run prod
```
